# go-spring-encryptors

Spring encryptors implemented in Go.
See https://docs.spring.io/spring-security/site/docs/current/api/org/springframework/security/crypto/encrypt/Encryptors.html

Supported encryptors:
 
- Standard (bytes)
- Stronger (bytes)
- Text (text)
- QueryableText (text)
- Delux (text)
